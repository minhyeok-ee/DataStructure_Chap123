# 2. \sigma (k=1,n)이 recursion으로 계산 과정을 그리시오

- base case : s(1) = 1
- recursion case : s(n) = n + s(n-1)

- n == 5 라고 가정하면 
```c
sum(5) = 5 + sum(4)
sum(4) = 4 + sum(3)
sum(3) = 3 + sum(2)
sum(2) = 2 + sum(1)
sum(1) = 1  (base case)
```

- 역으로 값이 반환됨
```c
sum(2) = 2 + 1 = 3
sum(3) = 3 + 3 = 6
sum(4) = 4 + 6 = 10
sum(5) = 5 + 10 = 15 (최종 결과)
```
